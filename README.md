# 起号学习看板 Chrome 扩展

这是一个 Manifest V3 本地扩展，用于滚动采集抖音关注瀑布列表里的账号信息，按 `config/categories.json` 中文关键词分类，缓存当前主页截图，并生成起号学习画廊。

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录：`chrome-chrome-edge-config-categories-json`

## 使用流程

1. 在 Chrome 里登录抖音，打开关注列表或其他账号列表页。
2. 打开关注列表弹窗，停留在“关注”标签。
3. 点击扩展图标，选择“开启实时滚动采集”。
4. 扩展会自动识别关注弹窗里的滚动容器，边滚动边采集，按主页链接去重合并。
5. 需要保留当前主页证据时，点击“缓存当前主页截图”。
6. 点击“打开离线画廊”，扩展会打开内置画廊页，并同步下载版 `data/gallery/index.html` 和 `data/gallery/accounts.json`。

## 关于本机登录态

- 最推荐：直接在你当前登录抖音的 Chrome 中加载本扩展，扩展会复用当前浏览器登录态。
- 如果希望使用独立持久化浏览器目录，可手动运行：

```bash
chmod +x scripts/open-chrome-debug.sh scripts/open-edge-debug.sh
./scripts/open-chrome-debug.sh 9222
```

然后在这个新打开的 Chrome 中登录抖音并加载本目录扩展。Edge 可用：

```bash
./scripts/open-edge-debug.sh 9223
```

Chrome 扩展本身不能启动浏览器或写回扩展目录，因此截图和下载版画廊通过浏览器下载 API 保存。若 Chrome 开启了“每次下载前询问保存位置”，请选择项目目录作为保存位置，才能得到本项目下的 `data/gallery/` 文件结构。扩展内置画廊会从浏览器本地存储读取最近采集的数据和最近 12 张截图预览。

## 分类规则

分类规则位于 `config/categories.json`。每个分类包含：

- `name`：文件夹/分组名称
- `keywords`：用于匹配昵称、简介、抖音号和页面可见文本的中文关键词

你可以直接编辑关键词后重新加载扩展。

## 已采集字段

- 昵称
- 简介
- 头像
- 主页链接
- 抖音号或页面可见 ID
- 分类
- 当前主页截图记录
- 原始页面可见文本片段

## 注意

抖音前端 DOM 会随版本变化，扩展使用的是“弹窗滚动容器 + 可见主页链接 + 附近文本”的采集策略。如果某些账号字段缺失，通常是当前列表项没有渲染该信息，重新打开关注弹窗后再次滚动采集即可。
